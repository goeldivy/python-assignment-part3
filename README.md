# Python Assignment - Part 3

## 📌 Project Overview
This project demonstrates the use of Python for handling JSON files and integrating APIs. It includes reading, writing, and modifying data, along with fetching real-time data from an online API.

---

## 🚀 Features
- Write student data into a JSON file
- Read data from a JSON file
- Calculate average marks of students
- Assign Pass/Fail status based on average
- Save updated data into a new JSON file
- Fetch user data from an online API

---

## 🧠 Concepts Used
- File Handling in Python
- JSON (json.load, json.dump)
- Lists and Dictionaries
- Loops and Conditional Statements
- String Formatting (f-strings)
- API Integration using requests module
